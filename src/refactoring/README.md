# Refactoring Techniques

Repository ini berisi contoh-contoh teknik refactoring berdasarkan kategori yang dikelompokkan oleh Martin Fowler dalam bukunya "Refactoring: Improving the Design of Existing Code".

Semua contoh dalam repository ini dibagi dalam dua package. `before` dan `after`. Sesuai dengan namanya, `before` adalah contoh code sebelum di-refactor, `after` adalah hasil setelah di-refactor.

Semua test case dalam repository ini menggunakan JUnit 5.

- [Moving Features between Objects](moving_features)
- [Organizing Data](organizing_data)
